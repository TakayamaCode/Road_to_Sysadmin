**El Reto "Master" (Mezclando todo): "El Instalador"

Imagina que vas a instalar una App.

1. Pide el nombre de usuario (Si no es **admin**, `exit 1`).
2. Pregunta: "¿Cuántos módulos quieres instalar?".
3. Crea una carpeta para cada módulo (`modulo_1`, `modulo_2`...).
4. **Dentro** de cada carpeta, crea un archivo vacío llamado `readme.txt`.
   
¿Cómo harías para que, además de crear el `readme.txt`, el script escribiera dentro de cada uno: _"Este es el modulo número $i"_?

Tu Reto de "Graduación" (El Desafío de Limpieza):

Como ya creaste muchas carpetas y archivos (`modulo_1`, `modulo_2`, etc.), ahora necesitas un script que **limpie el desastre**.

**Implementación: "El Desinstalador"**

1. Pregunta al usuario: "¿Qué prefieres: 1) Borrar todo o 2) Borrar un módulo específico?".
2. Si elige **1**: Usa un bucle `for` para borrar todas las carpetas que empiecen por `modulo_`. (Pista: `rm -rf modulo_*`).
3. Si elige **2**: Pregunta "¿Qué número de módulo quieres borrar?" y borra **solo esa carpeta** (ej: `modulo_3`).

## Reto "El panel profesional

1. **El Bucle Infinito:** Envuelve todo en un `while true; do ... done`.
2. **El Menú:** Muestra con `echo` tres opciones: `1. Instalar`, `2. Borrar`, `3. Salir`.
3. **La Captura:** Usa `read -p` para guardar la opción del usuario en la variable `OPCION`.
4. **El Selector (`case`):**
    - **Si es `1`**: Pide cuántos módulos (`read -p`), usa un `for` con `$(seq 1 $NUM)` para crearlos (`mkdir`) y ponles un `readme.txt`.
    - **Si es `2`**: Borra todo de un golpe (`rm -rf modulo_*`).
    - **Si es `3`**: Despídete y usa `break` para romper el bucle.
    - **Si es `*`**: Di que la opción no es válida.
